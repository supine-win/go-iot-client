# 测试与覆盖率

## 当前测试分层

- `mock/`：协议模拟服务端（用于集成测试）
- `*_test.go`：客户端行为、异常路径、重试与重连

## 本地执行命令

```bash
go test ./...
go test -race ./...
go vet ./...
```

## 覆盖目标

- 协议编解码单元测试
- 读写主流程集成测试
- 超时、断线、重连、错误码路径
- 并发读写与竞态检测（`go test -race`）

## 回归检查清单

- 修改协议编解码后：至少补 1 条正常路径 + 1 条异常路径测试
- 修改重试逻辑后：至少补断线恢复或错误注入测试
- 修改公开 API 后：同步更新 `doc/api/README.md` 示例
